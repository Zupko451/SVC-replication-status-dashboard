Connection Guide

Host — Management IP or FQDN of the FlashSystem / SVC node (e.g. 192.168.10.50 or flashsystem.corp.net).

Port — Default REST API port is 7443 on some firmware versions and 443 on others. Check your system's management console.

Username — Local or LDAP account with at least Monitor role on the array.

API Path — Most current FlashSystem firmware uses /rest/v1. Older SVC may use /rest/v1 as well; check IBM documentation for your firmware version.
